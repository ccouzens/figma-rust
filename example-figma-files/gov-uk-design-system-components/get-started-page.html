Getting set-up
Download GDS Transport
Use the published Assets
In order to use the styles and components, you’ll need to make sure you have GDS Transport font downloaded on your computer. If you’re not sure whether you should be using GDS Transport, read the Service Manual guide on making your service look like GOV.UK.

The next step is to use the published components and styles in a new figma file.

In your working file, you’ll be able to add the styles and components from this file using the Assets navigation tab next to the Layers navigation tab.
